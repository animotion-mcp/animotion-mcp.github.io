One Command Setup

Add to your MCP config 
{
  "mcpServers": {
    "animotion": {
      "command": "npx",
      "args": ["-y", "animotion-mcp"]
    }
  }
}

Where to paste: Claude Code → .claude/settings.json  |  Cursor → .cursor/mcp.json  |  Windsurf → MCP Settings

That's it. No clone, no install. 
# Or install globally
npm install -g animotion-mcp

# Or run directly
npx animotion-mcp
